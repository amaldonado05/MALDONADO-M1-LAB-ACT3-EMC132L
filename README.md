# M2 - Game Programming 2: 3D Obstacle Course

A 3D Unity obstacle course game showcasing player movement, environment design, hazards, and checkpoint-based respawning using C# scripting.

## Features
- Smooth WASD movement with jumping and running  
- Third-person camera view with mouse look  
- Challenging obstacle course with stairs and platforms.
- Death floors that trigger respawn at a checkpoint  
- Victory platform that displays a win message  
- On-screen fade text feedback (“YOU DIED!” / “YOU WON!”)

## How to Play
1. Move using WASD and look around with your mouse.  
2. Spacebar to jump, Left Shift to run.  
3. Avoid falling or touching red death floors — you’ll respawn at the checkpoint.  
4. Reach the other side to win the game!  

## Project Overview
- Demonstrates CharacterController-based movement, triggers, and respawn logic in Unity.  
- Uses TextMeshPro UI for feedback messages and simple fade animations.  
- Highlights fundamental gameplay mechanics: hazards, checkpoints, and victory conditions.
